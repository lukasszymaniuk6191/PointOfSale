# PointOfSale

## Prerequisites
jdk 8

## Running application
java -jar target/pointOfSale.jar 

## how to use application
1. Add new client - nc
2. Add new products to client - np
3. Add bar code and press Enter
  existing bar codes:
  "111-111-111"
  "222-222-222"
  "333-333-333"
  "444-444-444"
  "555-555-555"
4. Exit transaction - 'exit'
5. Exit application - 'exit'
